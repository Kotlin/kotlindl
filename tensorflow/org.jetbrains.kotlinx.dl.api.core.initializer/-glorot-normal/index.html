The Glorot normal initializer, also called Xavier normal initializer.

Draw samples from a truncated normal distribution centered on 0 with stddev = sqrt(2 / (fan_in + fan_out)) where fan_in is the number of input units in the weight tensor and fan_out is the number of output units in the weight tensor.

See also

Glorot et al., 2010
